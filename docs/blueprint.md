# **App Name**: FinanceFlow

## Core Features:

- Authentication: Secure user authentication using Firebase Authentication with Google and email/password login.
- Dashboard Overview: Display key financial metrics including total income, expenses, and balance via interactive cards.
- Categorized Expense Chart: Visualize spending patterns with a chart that breaks down expenses by category over different time periods.
- Transaction Management: Enable users to create, read, update, and delete financial transactions, including categorization and tagging.
- Automated Installment Handling: Automatically generate and manage installment transactions with customizable start dates and invoice assignments.
- Card and Invoice Management: Manage credit cards, track invoice details, and calculate available credit limits based on transactions.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) for a sense of trust and security.
- Background color: Light Gray (#F9FAFB) for a clean and modern feel.
- Accent color: Sky Blue (#38B6FF) to highlight key actions and information.
- Headline font: 'Space Grotesk', a modern sans-serif for headings; Body font: 'Inter', a grotesque-style sans-serif, for body text. The former gives a techy feel, while the latter makes for easy reading in longer blocks.
- Note: currently only Google Fonts are supported.
- Use clear and intuitive icons from a library like FontAwesome or Material Icons to represent categories and actions.
- Implement a responsive, grid-based layout that adapts seamlessly to different screen sizes.
- Incorporate subtle transitions and animations to provide feedback and enhance user engagement, such as loading spinners or confirmation alerts.
